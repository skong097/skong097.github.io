# MyPet 개발일지 — 2026-02-15

## 진행 사항

### Phase 1 Sprint 1: 기반 구축 시작

프로젝트 구조 생성 및 핵심 모듈 개발 완료.

**완료 항목:**

- **프로젝트 디렉토리 구조**: 계획서 Task 1.1 기반 전체 생성
- **config.py**: 카메라/제스처/FSM/렌더링/사운드/HUD 전역 설정값 정의
- **hand_tracker.py** (Task 1.2): MediaPipe 21 keypoints 추적, 궤적 버퍼(30프레임), 12종 제스처 인식 (palm_down/up, circle, swipe, fist_to_open, thumbs_up, wrist_wave, finger_walk 등)
- **gesture_classifier.py** (Task 1.3): 규칙 기반 분류기, 제스처별 Dwell 시간 정의, 신뢰도 스코어, 쿨다운 메커니즘
- **pet_controller.py** (Task 2.1 선행): FSM 12상태 정의 + 전이 규칙 + 감정 시스템 (happiness/excitement/tiredness/affinity)
- **pet_renderer.py** (Task 2.2 선행): Option A 도형 기반 프로토타입 (원+타원 조합), 상태별 애니메이션 (꼬리 흔들기/앉기/뛰기/짖기 이펙트)
- **ui_overlay.py** (Task 2.3 선행): HUD 상단바/하단 패널/감정 미니바/PIP/Dwell 진행바/명령 피드백
- **sound_manager.py**: pygame.mixer 기반 효과음, 파일 미존재 시 무음 처리
- **main.py**: 카메라→제스처→명령→FSM→렌더링 통합 루프 + 데모 모드(키보드)
- **requirements.txt**: Phase 1 종속성
- **README.md**: 실행 방법 + 제스처 매핑표

## 아키텍처 요약

```
Camera → HandTracker(MediaPipe) → GestureClassifier(Dwell)
  → PetController(FSM+Mood) → PetRenderer(도형) + UIOverlay(HUD)
```
