# 🐕 MyPet — 손 제스처 기반 가상 반려견 & 자율주행 로봇 제어 시스템

> **목표:** MediaPipe 손 제스처로 가상 강아지를 제어하고, 궁극적으로 4륜 자율주행 로봇에 탑재하여 실제 로봇견을 조작하는 통합 시스템

---

## 1. 프로젝트 개요

### 1.1 비전

사람의 손 동작(핸드 시그널)으로 가상/실제 로봇견을 제어하는 시스템. gaze_puzzle 프로젝트의 MediaPipe 손 추적 기술을 확장하여, 다양한 제스처를 인식하고 이를 강아지 행동 명령으로 매핑한다.

### 1.2 개발 단계

| Phase | 이름 | 목표 | 기간 |
|-------|------|------|------|
| Phase 1 | Virtual Pet | 화면 속 가상 강아지 + 손 제스처 제어 | 4~6주 |
| Phase 1.5 | Smart Pet | AR 오버레이 + 감정교감 + 강화학습 AI | 4~6주 |
| Phase 2 | Robot Bridge | ROS2 통합 + 시뮬레이션 환경 연동 | 4~6주 |
| Phase 3 | Real Robot | 4륜 로봇 탑재 + 듀얼 카메라 시스템 | 6~8주 |
| Phase 4 | Autonomous | 자율주행 + 명령 병행 제어 | 지속 |

## 2. 소프트웨어 스택

### 2.1 Phase 1 — Virtual Pet (현재 목표)

| 카테고리 | 기술 | 용도 |
|---------|------|------|
| 손 추적 | MediaPipe Hands | 21 keypoints 추적, 제스처 분류 |
| 렌더링 | OpenCV + Pygame (or PyGame CE) | 가상 강아지 2D 스프라이트 애니메이션 |
| 대안 렌더링 | Ursina Engine / Panda3D | 3D 강아지 모델 렌더링 (선택) |
| 제스처 분류 | 규칙 기반 → ML (Random Forest/LSTM) | 손 모양 + 동작 궤적 분류 |
| 애니메이션 | Sprite Sheet + State Machine | 강아지 행동별 애니메이션 전환 |
| 사운드 | pygame.mixer / playsound | 짖기, 꼬리흔들기 등 효과음 |
| 상태관리 | FSM (Finite State Machine) | 강아지 행동 상태 전이 관리 |

### 2.2 Phase 1.5 — Smart Pet (AR + 감정교감 + 강화학습)

| 카테고리 | 기술 | 용도 |
|---------|------|------|
| AR 합성 | OpenCV + ArUco / MediaPipe Objectron | 카메라 영상 위 바닥면 감지 + 강아지 합성 |
| 평면 추정 | cv2.solvePnP + 호모그래피 | 바닥 좌표계 추정, 3D→2D 투영 |
| 감정 인식 | EyeCon 파이프라인 재활용 | 주인 표정(미세표정/동공) → 감정 7종 분류 |
| 양방향 감정 | 감정 상태 머신 | 주인 감정 → 강아지 반응 매핑 |
| 강화학습 | Stable-Baselines3 (PPO/SAC) | 명령 수행 학습 에이전트 |
| 학습 환경 | Gymnasium (custom env) | 강아지 행동 시뮬레이션 환경 |
| 모델 저장 | torch + joblib | 학습된 정책 저장/로드 |

### 2.3 Phase 2 — Robot Bridge

| 카테고리 | 기술 | 용도 |
|---------|------|------|
| 미들웨어 | ROS2 Jazzy | 노드 통신, 토픽/서비스 |
| 시뮬레이션 | Gazebo Harmonic | 로봇 물리 시뮬레이션 |
| URDF/SDF | 로봇 모델링 | 4륜 로봇 + 카메라 마운트 정의 |
| Nav2 | 자율주행 스택 | 경로 계획, 장애물 회피 |
| 통신 | ROS2 Topics/Services | 제스처 → 로봇 명령 변환 |

### 2.4 Phase 3 — Real Robot (듀얼 카메라)

| 카테고리 | 기술 | 용도 |
|---------|------|------|
| 카메라 A | USB/CSI 카메라 | 사용자 손 제스처 인식 (전방) |
| 카메라 B | 광각 카메라 / Depth 카메라 | 주변 환경 인식 + 장애물 감지 |
| 하드웨어 | Jetson Orin Nano / RPi 5 | 온보드 추론 |
| 모터 제어 | micro-ROS + ESP32 | 4륜 구동 모터 제어 |
| SLAM | RTAB-Map / ORB-SLAM3 | 환경 매핑 + 위치 추정 |
| 객체 감지 | YOLOv8n / MobileNet | 주변 객체 인식 |

## 3. 기술 스택 상세

### 3.1 제스처 → 행동 매핑

```
┌─────────────────────────────────────────────────────────┐
│                    제스처 → 행동 매핑표                    │
├──────────────────┬──────────────────┬───────────────────┤
│     제스처        │    강아지 행동     │    인식 방식       │
├──────────────────┼──────────────────┼───────────────────┤
│ 손바닥 아래로 누름 │ 앉아 (Sit)       │ Palm down + 하강  │
│ 손바닥 위로 올림   │ 서 (Stand)       │ Palm up + 상승    │
│ 검지 원 그리기    │ 빙글빙글 (Spin)   │ 원형 궤적 추적     │
│ 손 좌/우 스와이프 │ 방향 전환 (Turn)  │ 수평 이동 벡터     │
│ 주먹 → 펼침      │ 짖어 (Bark)      │ 주먹→오픈 전이     │
│ 손가락 모아 입으로 │ 먹어 (Eat)       │ 핀치 + 위치       │
│ 양손 앞으로 밀기   │ 달려 (Run)       │ 양손 전진 동작     │
│ 손바닥 STOP      │ 멈춰 (Stay)      │ Open palm 정지    │
│ 박수             │ 잘했어 (Praise)   │ 양손 접근+분리 반복 │
│ 손목 흔들기       │ 이리와 (Come)     │ 손목 반복 동작     │
│ 검지+중지 걷기    │ 산책 (Walk)      │ 두 손가락 교대     │
│ 엄지 척          │ 좋아 (Good boy)   │ 엄지 up          │
└──────────────────┴──────────────────┴───────────────────┘
```

### 3.2 강아지 FSM (Finite State Machine)

```
                    ┌─────────┐
              ┌────→│  IDLE   │←────┐
              │     └────┬────┘     │
              │          │          │
         [Stay]    [Sit/Stand]   [timeout]
              │          │          │
              │     ┌────▼────┐     │
              │     │   SIT   │─────┘
              │     └────┬────┘
              │          │[Run]
              │     ┌────▼────┐
              ├─────│   RUN   │──[Turn]──→ TURN
              │     └────┬────┘           │
              │          │                │
              │     [Spin]               [done]
              │          │                │
              │     ┌────▼────┐          │
              └─────│  SPIN   │←─────────┘
                    └────┬────┘
                         │[Bark]
                    ┌────▼────┐
                    │  BARK   │──[Eat]──→ EAT
                    └─────────┘
```

## 5. Phase 1.5 상세 작업 계획 (Smart Pet — AR + 감정교감 + 강화학습)

> Phase 1 완료 후 진행. 가상 강아지를 실제 카메라 영상 위에 합성하고, 주인과 양방향 감정 교감을 하며, 스스로 학습하는 AI 강아지로 진화시킨다.

### Sprint 4: AR 오버레이 (Week 7~8)

**Task 4.1 — 바닥면 감지 + 좌표계 설정**
- ArUco 마커 기반 바닥 평면 추정 (초기 버전)
- 마커 없는 평면 추정으로 전환 (MediaPipe Objectron 또는 호모그래피)
- 카메라 캘리브레이션 → 3D 좌표 → 2D 투영 파이프라인

**Task 4.2 — 강아지 AR 합성 렌더링**
- 바닥면 위에 강아지 스프라이트 배치 (원근감 적용)

- 강아지 크기를 거리에 따라 자동 조절 (원근 투영)
- 카메라 영상 위 반투명 합성 (cv2.addWeighted)

### Sprint 6: 강화학습 AI 강아지 (Week 11~13)

**Task 6.1 — Gymnasium 커스텀 환경 설계**
```python
# mypet/rl/pet_env.py
class PetEnv(gymnasium.Env):
    """
    관찰 공간 (observation):
        - 주인 제스처 원핫 (12차원)
        - 주인 감정 벡터 (7차원)
        - 강아지 현재 상태 (12차원)
        - 강아지 위치 (x, y)
        - 친밀도 (1차원)
        - 시간 경과 (1차원)
        → 총 35차원

    행동 공간 (action):
        - 12가지 행동 (Discrete(12))
        - SIT, STAND, RUN, WALK, SPIN, BARK,
          EAT, TURN, COME, STAY, IDLE, PRAISE_REACT

    보상 함수:
        - 올바른 명령 수행: +10
        - 감정에 맞는 반응: +5
        - 빠른 반응 (< 0.5초): +3 보너스
        - 명령 무시: -5
        - 부적절한 반응: -3
        - 주인 친밀도 상승: +2
    """
```

**Task 6.2 — PPO 학습 파이프라인**
- Stable-Baselines3 PPO 에이전트 설정
- 사전 학습 (Pretrain): FSM 규칙 기반 데모 → 행동 복제 (BC)
- 미세 조정 (Fine-tune): 실제 인터랙션 데이터로 PPO 학습
- 학습 스케줄:
  - Stage A: 규칙 기반 데모 1만 에피소드 → BC로 초기 정책
  - Stage B: PPO 자기 학습 5만 스텝 → 기본 명령 수행
  - Stage C: 감정 반응 보상 추가 → 공감 행동 학습
  - Stage D: 온라인 학습 (실시간 유저 피드백)

**Task 6.3 — 성장 시스템 통합**
- 학습 진행도 시각화 (경험치 바, 레벨)
- 새로운 행동 해금 (레벨업 시 트릭 추가)
- 성격 분화: 학습 데이터에 따라 강아지 성격이 달라짐
  - 활발한 강아지 vs 차분한 강아지
  - 장난꾸러기 vs 순종적
- 모델 저장/로드 → 내 강아지 고유 개성 유지

## 6. 추가 아이디어 (향후 검토)

### 6.1 음성 명령 병행 (Voice + Gesture)

손 제스처에 음성 명령을 결합하면 더 자연스러운 인터랙션이 가능합니다.

- 기술: Whisper (STT) + 키워드 감지
- 예시: "앉아" + 손바닥 내리기 → 더 빠른 반응
- 장점: 제스처 인식 실패 시 음성으로 보완

### 6.2 멀티 펫 시스템

여러 마리의 가상 강아지를 동시에 제어합니다.

- 양손 각각 다른 강아지 지정
- 왼손: 강아지 A, 오른손: 강아지 B
- 팩(Pack) 명령: 양손 동시 제스처 → 전체 명령

### 6.3 훈련 모드 (Tutorial + Gamification)

게임화된 훈련 시스템으로 사용자와 강아지 모두 학습합니다.

- 미션: "5초 안에 앉아 명령 3회 성공" → 별 획득
- 레벨업: 새로운 제스처/명령 해금
- 도감: 수행한 트릭 컬렉션

### 6.4 LLM 통합 자연어 대화

Home Guard Bot의 guard_brain 경험을 활용합니다.

- 기술: Ollama + EXAONE / Llama
- 기능: "우리 강아지 오늘 기분 어때?" → 상태 기반 자연어 응답
- TTS 연동: 강아지 상태를 음성으로 알려줌
