## 1. 작업 요약

Phase 1(v3.1)에서 구축한 4분할 대시보드 기반 위에
영상 6개 + 음성 7개 = 13개 심리 지표 실시간 추출 파이프라인 완성,
Blendshapes 기반 7감정 분류 + 멀티모달 복합 스코어(영상 60% + 음성 40%) 구현.
내장 마이크 설정, LLM 응답 속도 최적화, 썬글라스 오버레이까지 완료.

## 3. Phase 2 변경사항

### 3.1 Phase 2A — 영상 지표 고도화 (`core/vision.py`)

| 지표 | 구현 내용 |
|------|----------|
| 미세 표정 | Blendshapes 급변 감지 (0.5초 내 delta>0.3), 유형 라벨링 (억제된 웃음, 미간 찌푸림, 걱정/놀람 등) |
| 동공 확장률 | 최초 10초(300프레임) 자동 베이스라인 → 이후 변화율 % 표시 |
| 시선 패턴 | 현재 방향 고정 시간(초) + 최근 60초 회피 횟수 추적 |
| 비대칭 종합 | 6개 좌/우 AU 쌍 (smile, frown, browDown, browOuterUp, eyeSquint, cheekSquint) 평균 차이 |
| 입술 압축 | FaceData에 `lip_press_score` 직접 포함 |

**FaceData 추가 필드:**
- `pupil_dilation_pct` — 베이스라인 대비 변화율 (%)
- `gaze_fixation_sec` — 현재 방향 고정 시간
- `gaze_aversion_count` — 최근 60초 시선 회피 횟수
- `micro_expression_count` — 미세 표정 감지 건수
- `micro_expression_label` — 마지막 감지 유형
- `asymmetry_score` — 종합 비대칭 점수
- `lip_press_score` — 입술 압축 점수

### 3.2 Phase 2B — 음성 파이프라인 연결

**데이터 흐름:**
```
TTS 종료 → _tts_end_time 기록
         → 0.5초 후 STT 시작
         → timing_signal → _stt_start_time 기록
         → audio_signal → WAV 경로 임시 저장
         → result_signal → 텍스트 수신
         → VoiceAnalyzer.analyze(WAV + 텍스트 + 응답지연)
         → _latest_voice_data 업데이트
         → 지표 패널 실시간 표시
```

**음성 지표 7개:**
| 지표 | 분석 방법 |
|------|----------|
| 피치 (F0) | librosa YIN 알고리즘 (75~500Hz) |
| Jitter | 연속 피치 주기 간 변동률 (%) |
| Shimmer | RMS 기반 진폭 변동률 (%) |
| 응답 지연 | TTS 종료 → STT 발화 시작 타임스탬프 차이 |
| 말 속도 | 글자수(음절수) / 발화 시간 (음절/초) |
| 음량 | RMS 평균/표준편차/범위 |
| 비유창성 | STT 텍스트에서 "음", "어", "아" 등 필러 + 단어 반복 감지 |

### 3.3 내장 마이크 설정

**문제:** USB 마이크에서 노트북 내장 DMIC로 전환 시 인식 불가
**원인:** PipeWire가 DMIC(card 3: acp6x)를 점유 + 배경 소음으로 에너지 임계값 자동 상승

**해결:**
- `settings.py` STT 설정에 마이크 장치/에너지 임계값 수동 고정 추가
- `stt_engine.py`에 `device_index`, `energy_threshold`, `pause_threshold` 지원

```python
STT = {
    "microphone_index": 10,        # PipeWire 장치
    "energy_threshold": 15000,     # 수동 고정 (내장 DMIC용)
    "dynamic_energy": False,       # 자동 조절 비활성화
    "pause_threshold": 1.5,        # 1.5초 침묵이면 발화 종료
    "phrase_time_limit": 15,       # 최대 발화 시간 15초
}
```

**진단 도구:** `check_mic.py` — 장치 목록 / 개별 테스트 / 자동 적용

### 3.4 LLM 응답 속도 최적화

**문제:** 사용자 발화 후 AI 답변까지 체감 지연이 김
**원인:** 매 호출마다 긴 프롬프트(성격/말투 ~300자) 반복 전송 + 응답 길이 무제한

**해결 (`core/llm_client.py` + `settings.py`):**
- 시스템 프롬프트 분리 — `system` 파라미터로 별도 전송, `prompt`는 데이터만
- `num_predict: 80` — 최대 생성 토큰 80으로 제한
- `temperature: 0.7`

```python
# 프롬프트: 기존 ~300자 → 현재 ~30자
"[스트레스: 25] 사용자: '오늘 좀 피곤해요'"
```

### 3.5 썬글라스 오버레이 (`core/vision.py` draw_landmarks)

**구성:**
- 렌즈: 노란색 반투명 사다리꼴 (위가 넓은 형태)
- 크기: 양쪽 홍채 사이 거리 기반 고정 비율 (가로 0.29, 세로 0.18)
- 브릿지: 코 랜드마크(6번) 경유 곡선
- 템플: 관자놀이(234, 454번)까지 연장
- 홍채: 틴트 위 초록 점 추적
- 반사광: 좌상단 하이라이트

## 4. Phase 3 변경사항

### 4.1 7감정 분류 (`core/analyzer.py`)

**Blendshapes 규칙 엔진:**

| 감정 | 주요 AU/Blendshapes | 가중치 |
|------|---------------------|--------|
| Happy 😊 | mouthSmile (60%) + cheekSquint (40%) | — |
| Sad 😢 | mouthFrown (50%) + browInnerUp (50%) | — |
| Angry 😠 | browDown (40%) + mouthPress (30%) + jawForward (30%) | — |
| Surprise 😲 | eyeWide (30%) + browInnerUp (20%) + browOuterUp (20%) + jawOpen (30%) | — |
| Fear 😰 | browInnerUp (30%) + eyeWide (30%) + mouthPress (20%) + lipPress (20%) | — |
| Disgust 😖 | noseSneer (50%) + mouthShrugUpper (30%) + mouthFrown (20%) | — |
| Neutral 😐 | 0.5 - max(다른 감정) | — |

→ softmax 정규화 후 최고 점수 감정 선택

### 4.2 멀티모달 복합 스코어

**영상 스코어 (6개 지표 × 가중치):**
| 지표 | 가중치 | 스코어 변환 |
|------|--------|------------|
| 홍채 불안정도 | 0.15 | 직접 사용 (0~100) |
| 깜빡임 변화율 | 0.15 | 정상 범위(15~20) 대비 편차 % |
| 동공 확장률 | 0.20 | abs(변화%) × 5 |
| 미세 표정 | 0.20 | 건수 × 20 (5건=100) |
| 비대칭 | 0.15 | 임계값 대비 비율 × 50 |
| 입술 압축 | 0.15 | 임계값 대비 비율 × 50 |

**음성 스코어 (6개 지표 × 가중치):**
| 지표 | 가중치 | 스코어 변환 |
|------|--------|------------|
| 피치 변화 | 0.20 | F0 std × 2 |
| 응답 지연 | 0.20 | 지연 시간 기반 단계별 |
| 비유창성 | 0.15 | 건수 × 25 |
| Jitter | 0.15 | jitter% × 10 |
| 말 속도 변화 | 0.15 | 정상(4음절/초) 대비 편차 % |
| 음량 변화 | 0.15 | volume_std × 500 |

**퓨전:** `combined = vision × 0.6 + voice × 0.4`
(음성 없으면 영상만 100% 사용)

### 4.3 이상 감지 플래그

자동 감지 조건:
- 시선 회피 빈번 (60초 내 8회 초과)
- 깜빡임 급증 (정상 상한 × 1.5)
- 미세 표정 다발 (60초 내 5건 이상)
- 입술 압축 강함 (임계값 × 1.5)
- 응답 지연 과다 (5초 초과)
- 비유창성 과다 (4건 이상)

### 4.4 베이스라인 시스템 (`core/baseline.py`)

- 대화 시작과 동시에 30초간 자동 수집
- 영상: EAR, iris_ratio, iris_instability, blink_rate, asymmetry, lip_press
- 음성: f0_mean, f0_std, speech_rate, volume_mean, jitter_pct
- 수집 완료 시 콜백으로 Analyzer에 자동 전달
