## 🐛 **문제 해결**

### **Issue 1: PyTorch 2.6 weights_only 오류**

**에러 메시지:**
```
WeightsUnpickler error: Unsupported global: GLOBAL numpy.core.multiarray.scalar 
was not an allowed global by default.
```

**원인:**
- PyTorch 2.6부터 `torch.load()`의 기본값이 `weights_only=True`로 변경됨
- numpy 배열이 포함된 체크포인트 로드 시 보안 오류 발생

**해결:**
```python
# Before
checkpoint = torch.load(model_path, map_location=self.device)

# After
checkpoint = torch.load(model_path, map_location=self.device, weights_only=False)
```

**수정 파일:** `stgcn_inference.py` (70줄)
