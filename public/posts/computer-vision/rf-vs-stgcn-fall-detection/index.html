배경#

낙상 감지에 어떤 모델이 최적일까? 이 질문에 답하기 위해 두 가지 접근법을 직접 비교했습니다.

  • Random Forest: 프레임 단위 특징 추출 → 즉시 분류
  • ST-GCN: 시간적 골격 그래프 → 시퀀스 기반 분류

정확도만 보면 RF가 높지만, 실제 운영에서는 그렇게 단순하지 않았습니다.
