## 개요

**EyeCon 피노키오 멀티모달 심리 분석 AI**는 실시간으로 사용자의 얼굴 표정, 홍채 움직임, 눈 깜빡임 등 비언어적 신호와 음성 패턴을 분석하여 심리 상태와 거짓말 가능성을 평가하는 혁신적인 대화형 AI 시스템입니다. 본 프로젝트는 NVIDIA RTX 4050 그래픽 카드와 노트북 카메라, 마이크, 스피커를 활용하여 개발되었으며, 한국어 환경에서의 사용자 친화적인 인터페이스를 제공합니다.

---

## 배경

심리 상태 분석과 거짓말 탐지 분야에서 비언어적 의사소통의 중요성이 점차 부각되고 있습니다. EyeCon은 이러한 요구에 부응하기 위해 설계되었습니다. 이 시스템은 MediaPipe Face Mesh와 홍채 추적 기술을 기반으로 얼굴의 미세한 변화를 감지하고, librosa와 WebRTC Vad를 통해 음성의 피치, 떨림, 응답 지연 등을 분석합니다. 특히, 이 프로젝트는 멀티모달 데이터 융합을 통해 심리 상태를 종합적으로 평가하고자 합니다. 초기 버전인 v2.6에서는 실시간 얼굴 랜드마크 추적, 스트레스 스코어 계산, 음성 분석 기능 등 핵심 기능들이 구현되었으나, 아직 개선의 여지가 많습니다.

---

## 구현

### 현재 상태

EyeCon은 현재 다음과 같은 주요 기능을 구현하고 있습니다:

- **영상 분석:**
  - MediaPipe Face Mesh를 활용한 478개 랜드마크 추적 (홍채 포함)
  - 홍채 좌표 추적 및 스트레스 스코어 계산
  - PyQt6 기반 GUI를 통한 실시간 데이터 시각화

- **음성 분석:**
  - Edge TTS를 통한 한국어 음성 출력
  - Google Speech Recognition을 통한 음성 입력 처리
  - librosa와 WebRTC Vad를 통한 피치, 떨림, 응답 지연 분석

- **대화 및 분석:**
  - Ollama LLM과 EXAONE 3.5를 통합한 심리 분석 대화 시스템
  - 실시간 대시보드를 통한 데이터 시각화

### 한계점

현재 구현 단계에서는 다음과 같은 한계점이 존재합니다:

- **단일 지표 의존:** 스트레스 스코어가 홍채 움직임만 반영하여 표정, 깜빡임, 동공 크기 등을 고려하지 않음
- **감정 분류 부재:** Happy, Sad, Angry 등 구체적인 감정 분류가 미흡
- **미세 표정 분석 미구현:** AU(Action Unit) 기반 미세 표정 감지 기능 부재
- **깜빡임 빈도 미추적:** 눈 깜빡임 빈도가 거짓말 탐지의 중요한 지표임에도 불구하고 누락
- **동공 크기 측정 미구현:** 동공 확장이 거짓말 탐지에 중요한 지표임에도 불구하고 측정되지 않음
- **베이스라인 부족:** 개인별 정상 상태 기준치가 없어 절대값으로 판단
- **LLM 프롬프트 단순화:** 다차원 데이터를 포함한 복잡한 분석 결과 전달 미흡
- **모듈화 부족:** 모든 로직이 `main.py`에 집중되어 있어 유지보수 어려움

## 결과

### 성과 지표

- **구현 완료 기능:**
  - MediaPipe Face Mesh 및 홍채 추적 (✅)
  - 스트레스 스코어 계산 (✅)
  - PyQt6 GUI 및 실시간 데이터 시각화 (✅)
  - Edge TTS 음성 출력 및 Google Speech Recognition 음성 입력 (✅)
  - Ollama LLM 대화 시스템 (✅)

- **개선 방향:**
  - 감정 분류 모델 도입을 통한 정확성 향상
  - 미세 표정 분석 및 깜빡임 빈도 추적 기능 추가
  - 동공 크기 측정 및 개인별 베이스라인 학습 기능 구현
  - 다차원 데이터를 고려한 복합 스코어 계산 알고리즘 개선

### 예시 성과

- **스트레스 스코어 계산 예시:** 홍채 움직임의 표준 편차를 기반으로 스트레스 스코어를 산출하여 사용자의 현재 스트레스 수준을 추정합니다.
- **음성 분석 예시:** 피치 변화와 응답 지연 시간을 분석하여 사용자의 심리적 압박감을 평가합니다.
