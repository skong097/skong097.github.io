## 개요

**"Gaze Puzzle: 미디어파이브 손 제스처 숨은그림찾기 게임"**은 사용자의 손 제스처만으로 조작 가능한 혁신적인 비전 AI 게임입니다. MediaPipe Hands 기술을 기반으로 검지 포인팅을 통해 사진 속 미묘한 차이점을 찾아내는 재미를 선사합니다. 키보드 없이도 웹캠을 통해 손의 움직임만으로 게임을 즐길 수 있으며, 다양한 난이도의 퍼즐 모드를 통해 사용자에게 도전적인 경험을 제공합니다.

---

## 배경

이 게임은 디지털 인터랙티비티와 인공지능 기술의 융합을 목표로 개발되었습니다. 특히, **MediaPipe Hands** 라이브러리를 활용하여 정확한 손 관절 추적과 제스처 인식을 구현하였습니다. 이를 통해 사용자는 검지를 이용해 차이점을 가리키거나, 손바닥 뒤집기로 게임을 전환하는 등 직관적인 손 동작으로 게임을 조작할 수 있게 되었습니다. 게임의 핵심은 **초급 모드**에서는 코믹한 낙서를 추가하여 차이점을 쉽게 찾을 수 있도록 설계되었고, **중급 모드**에서는 YOLO 객체 감지 기술을 활용해 더욱 복잡하고 자연스러운 차이점을 생성하여 사용자에게 더 큰 도전을 제공합니다.

---

## 구현

### 기술 스택

- **MediaPipe**: 손 관절 추적 및 제스처 인식
- **OpenCV**: 영상 처리 및 게임 렌더링
- **NumPy (<2)**: 수치 연산 지원 (MediaPipe 호환 필수)
- **Ultralytics**: YOLO v8 객체 감지 (중급 모드)

### 파일 구조

```
gaze_puzzle/
├── main.py               # 게임 실행 진입점
├── hand_tracker.py        # MediaPipe Hands 검지 추적 및 제스처 인식
├── puzzle_generator.py    # 초급 및 중급 퍼즐 생성기
├── requirements.txt       # Python 패키지 목록
├── README.md              # 문서화 파일
└── assets/                # 퍼즐 이미지 (jpg/png 자동 인식)
    ├── 01_marathon.jpg
    ├── 02_hobbit_door.jpg
    └── ...
```

### 환경 설정

게임을 실행하기 위해 다음 명령어를 사용합니다:

```bash
cd gaze_puzzle
python3 -m venth gaze_venv  # 가상 환경 생성
source gaze_venv/bin/activate  # 활성화 (Windows: gaze_venv\Scripts\activate)
pip install -r requirements.txt  # 필요 패키지 설치
```

### 실행 방법

게임을 시작하려면 다음 명령어를 실행합니다:

```bash
source gaze_venv/bin/activate
python main.py
```

웹캠이 연결되어 있어야 하며, 게임 창이 열리면 손을 카메라에 보여주세요.

## 조작법

### 손 제스처 조작

| 제스처          | 동작                          | 유지 시간   | 상태       |
|-----------------|-------------------------------|-------------|------------|
| ☝️ 검지 포인팅   | 차이점 가리키기              | 0.8초 유지  | 플레이 중   |
| 👋 손바닥 뒤집기 | 새 게임 / 다음 레벨          | 0.5초 유지  | 준비 완료, 클리어 |
| 👊 주먹 쥐기     | 현재 레벨 재시작              | 1.5초 유지  | 플레이 중   |
| 🤞 검지+중지 교차 | 게임 종료                    | 2.0초 유지  | 항상       |

- **포인팅 판별**: 검지만 펴고 중지를 접으면 초록 커서가 활성화되고, 그렇지 않으면 주황 커서가 표시됩니다.
- **플립 제스처 흐름**: 손바닥을 보이게 한 후 "PALM..." 메시지가 표시되고 "FLIP NOW!" 메시지가 나타나면 손을 뒤집습니다.

### 키보드 보조 조작

| 키       | 동작                   |
|----------|------------------------|
| N        | 새 게임 / 다음 레벨    |
| R        | 현재 레벨 재시작       |
| Q        | 종료                   |
| M        | 난이도 모드 전환 (초급 ↔ 중급) |

## 게임 모드

### 초급 (EASY) — 코믹 낙서

- **낙서 종류**: 콧수염, 뿔, 왕관, 하트 등 14가지 낙서 추가
- **차이점 수**: 레벨 1에서 3~4개, 레벨 2에서 4~6개, 레벨 3 이상에서 5~8개
- **낙서 크기**: 레벨에 따라 크기 조절 (25~40px, 18~32px, 12~22px)

### 중급 (MEDIUM) — YOLO 객체 복제 + 사진 영역 변형

- **YOLO 감지 성공 시**: 객체 복제 변형 (clone_shift, clone_flip, clone_scale, remove)
- **YOLO 감지 실패 시**: 영역 변형 (region_clone, region_remove, region_color, region_flip, texture_swap)
