## 2. 블로그 3D 히어로 — 우주 배경 효과 추가 ✅

**파일**: `layouts/partials/hero-3d.html`

### 기존 문제
- 배경이 순수 블랙(#050508)으로 단조로움

### 추가된 CSS 배경 레이어

| 레이어 | 위치 | 색상 | 투명도 |
|--------|------|------|--------|
| 오로라 커튼 | 좌상단 25%,10% | 초록/시안 | 0.2 |
| 태양 반사광 | 우상단 85%,15% | 골드/앰버 | 0.14 |
| 퍼플 네뷸러 | 중앙 50%,55% | 퍼플/인디고 | 0.18 |
| 딥블루 | 좌하단 15%,80% | 네이비 | 0.22 |
| 마젠타 힌트 | 우하단 80%,75% | 마젠타 | 0.1 |
| 렌즈 플레어 3개 | 분산 | 골드/블루/퍼플 | 0.08~0.1 |

### CSS 애니메이션
- `auroraShift`: 20초 주기 hue-rotate + brightness 변화 — 오로라 색상 이동
- `nebulaGlow`: 15초 주기 opacity + scale 변화 — 성운 호흡

### Three.js 추가 오브젝트

| 오브젝트 | 수량 | 크기 | 투명도 |
|----------|------|------|--------|
| 배경 별빛 | 1200개 | 0.05 | 0.8 |
| 성운 파티클 (5색) | 60개×5 = 300개 | 0.25~0.4 | 0.1~0.18 |

- 성운 색상: 초록(0x10b981), 블루(0x60a5fa), 퍼플(0xa78bfa), 골드(0xfbbf24), 인디고(0x818cf8)
- Fog 밀도: 0.015 → 0.008 (원거리 별빛 가시성 확보)
- 별빛 미세 회전 애니메이션 추가

### 투명도 튜닝 과정
1. 최초: 희미 (0.03~0.12) → 분위기 부족
2. 강화: 강함 (0.15~0.3) → 성운 멋지지만 약간 과함
3. **최종**: 중간값 (0.1~0.22) → 은은하면서 깊이감 있는 균형
